# 常识，如何获得常识？

> 通的方法是常识 — 金岳霖《知识论》

## 问题背景

### 我们面临的困境

!!! warning "理论泛滥的互联网"
    在信息爆炸的时代，我们被各种理论、观点、方法论包围。但真正的"常识"却越来越稀缺。

### 人的现状：身体与心智的分离

现代人常常处于一种**悬浮状态**：

- 大脑被抽象概念填满
- 身体却缺乏真实体验
- 理论与实践脱节

## 核心问题

### 什么是常识？

常识不是：

- 不是书本上背诵的知识
- 不是道听途说的信息
- 不是未经思考的默认观点

!!! success "常识是"
    - 通过身体力行获得的真知
    - 能够指导实际行动的判断力
    - 在具体情境中做出恰当反应的能力

### 常识与研究型学习

!!! question "矛盾？"
    - 获取常识往往是**无意识**的
    - 研究型学习似乎是**有意识**的
    
    两者如何统一？

## 获得常识的路径

### 1. 概念考察

在做任何研究之前，先搞清楚：

- 这个概念到底是什么意思？
- 它在什么语境下使用？
- 它与其他概念有什么关系？

### 2. 研究型学习

普遍存在的研究型学习本身就是获取常识的过程：

!!! example "例子"
    **获取恋爱/他人的常识**：
    - 不是看多少"恋爱理论"
    - 而是实际与人相处、观察、反思

### 3. 对比：ChatGPT 有常识吗？

!!! info "AI 的局限"
    ChatGPT 可以输出大量信息，但它：
    
    - 没有身体经验
    - 无法验证信息的真实性
    - 缺乏具体情境的判断力
    
    这正是"常识"的核心价值所在。
