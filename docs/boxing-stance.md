# 拳击站姿

拳击的站姿是一切技术的基础。正确的站姿能够提供稳定性、移动灵活性和出拳准备。

## 下半身

### 脚的位置

| 要点 | 说明 |
|------|------|
| **双脚距离（左右）** | 约一拳宽度 |
| **双脚距离（前后）** | 约膝盖距离 |
| **后脚位置** | 后脚脚跟与前脚脚尖在同一直线上 |
| **脚尖朝向** | 45° 外展 |

### 重心

!!! tip "重心分配"
    - **55%～60%** 的体重在后脚
    - 或**平分**体重
    - **切忌前倾**

### 蹲姿

- 蹲得**深一些**：重心低，移动更灵活
- 后脚脚跟**离地**：便于快速移动和旋转

## 上半身

### 身体角度

- 身体 **45°** 侧向对手
- 减少暴露面积
- 便于旋转发力

### 手的位置

!!! warning "防守要点"
    - **手肘压在肋骨上**：保护侧腹
    - **后手放在下巴上方**
    - **前手位置**：「六加五」位置（约面部前方）
    - **手腕内扣**：保护手腕

### 头部

- **收下巴**：保护下颌
- **眼睛**：盯着对手的胸部或肩部，而非眼睛（可以看到全身动作）

---

## 训练建议

1. **每天练习站姿**：肌肉记忆需要重复
2. **照镜子检查**：确保动作标准
3. **保持放松**：紧张会降低反应速度
4. **移动中保持**：站姿不是静态的，移动时也要保持
