# 欢迎来到我的个人网站

### 关于这个空间

这里记录着我的学习、思考与事情。我目前把内容归到三个核心区域：

| 区域 | 内容 | 特点 |
|------|------|------|
| **知识** | 原子化笔记、概念整理 | 这是一个不断生长、演化的个人知识区域 |
| **文章** | 方法、心理与经验反思 | 思考，梳理 |
| **项目** | 我目前在做、做过的有意思的事情 | 包含主题研究与菜单等生活实验 |

!!! quote "螺旋上升"
    知识帮助我沉淀，文章帮助我组织，项目则驱动做事的过程。

### 为什么选择网站作为个人装置？

- **所有权**：不依托任何平台，这是独属于自己的一片天地；
- **实验性**：在这里试验一种理想的个人生活组织形式；
- **负责任**：将自己的生活编织在社会网络中，让自己更负责任。
